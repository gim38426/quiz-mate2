⚡ 퀴즈메이트 (QuizMate) - 강의자료 기반 시험 대비 퀴즈 생성기
**퀴즈메이트(QuizMate)**는 대학생과 학습자들이 전공 강의 노트나 공부 자료를 입력하면, 해당 텍스트를 즉석에서 분석하여 퀴즈(객관식, OX 진위형, 빈칸 주관식)를 생성하고 풀이할 수 있도록 돕는 웹 기반 자율 학습 도구입니다.

서버가 필요 없는 100% 클라이언트 사이드 웹 애플리케이션으로, HTML, CSS, JavaScript만을 사용하여 제작되어 GitHub Pages를 통해 별도의 요금이나 설치 과정 없이 정적 웹페이지로 간편하게 호스팅 및 배포할 수 있습니다.

🌟 주요 기능
학습 내용 기반 자동 퀴즈 빌더:

입력된 강의 노트 텍스트를 문장 단위로 파싱하고 가중치(글자 수, 영문/기술 용어 매칭 등)가 부여된 핵심 명사를 탐색하여 최적의 문제를 생성합니다.
다양한 문제 유형 지원:

객관식 (4지선다): 본문 내 주요 핵심어와 본문의 다른 영역에서 추출한 키워드들을 무작위로 혼합하여 오답 보기를 지능적으로 생성합니다.
OX 진위형: 원래 문장에 적힌 내용의 참(O) 또는 핵심 키워드를 다른 단어로 변환한 거짓(X) 문장을 제작하여 이해도를 평가합니다.
빈칸 주관식 (단답형): 문장에서 핵심어 부위를 빈칸으로 전환하여 사용자가 답안을 타이핑하도록 만듭니다. 사용자가 제출한 정답 비교 시 불필요한 대소문자나 앞뒤 띄어쓰기 공백을 제거하고 정확하게 대조합니다.
혼합형: 3가지 유형을 골고루 배정하여 전방위적인 이해력을 검증합니다.
반응형 대화식 UI:

풀이 진행 중 진행률(Progress Bar)이 가변적으로 채워지며, 답안 제출 즉시 정답 유무 피드백과 본문의 원래 문장(해설)을 함께 시각적으로 제공합니다.
최종적으로 전체 정답률, 문항별 점수, 맞힌 개수를 원형 컨테이너 챠트로 깔끔하게 요약해 보여줍니다.
스마트 오답노트 보존 (LocalStorage):

오답 발생 시, 과목명, 문제 내용, 내가 고른 답, 실제 정답, 출처 해설을 브라우저 로컬 저장소(localStorage)에 저장합니다.
브라우저를 새로고침하거나 창을 닫아도 기록이 유지되며, 과목별로 오답을 필터링해 볼 수 있는 조회 기능 및 개별/전체 삭제 기능(CRUD)을 내장하고 있습니다.
🛠️ 기술적 특징 및 알고리즘 구현 방식
무서버 & 무외부 라이브러리(Zero-Dependency): 외부 API(OpenAI, Gemini 등) 키 유출 위험이나 API 제한이 발생하지 않도록 100% 브라우저 기반 순수 자바스크립트만 사용합니다.
한국어 조사 제거 휴리스틱 (Keyword Cleaning):
"한국어는 조사가 풍부하여 블라인드 처리를 하면 서울은(는)처럼 보기가 부자연스러워지는 단점이 있습니다."
이를 극복하기 위해 은, 는, 이, 가, 을, 를, 의, 에, 와, 과, 로, 으로, 에서, 도, 만, 이다, 합니다 등의 빈출 조사를 매칭하여 단어 끝부분에서 깔끔하게 정제해 내는 정규식 규칙 기반 접미사 스트리퍼(Suffix Stripper) 알고리즘을 사용합니다.
conic-gradient 원형 차트: 별도의 차트 라이브러리(Chart.js 등) 없이 CSS3의 원뿔형 그라디언트를 활용해 가볍고 세련된 결과 도표를 동적으로 렌더링합니다.

🚀 실행 및 테스트 방법
로컬 환경에서 실행하기
이 프로젝트의 폴더(quizmate)를 다운로드하거나 Clone 합니다.
폴더 내의 index.html 파일을 더블 클릭하여 웹 브라우저(Chrome, Edge, Safari 등)에서 바로 열어 실행합니다.
별도의 npm install이나 서버 세팅을 요구하지 않으므로 즉시 브라우저에서 사용할 수 있습니다.
🌐 GitHub Pages 배포 방법
GitHub을 사용해 이 사이트를 인터넷 상에 무료로 무료 호스팅하는 단계입니다:

GitHub 저장소(Repository) 생성:

본인의 GitHub 계정에 새 저장소(예: quizmate)를 개설합니다. (Public 저장소여야 합니다.)
파일 업로드 및 커밋 (Command Line 예시):

bash

git init
git add index.html style.css script.js README.md
git commit -m "Initial commit: QuizMate Release 1.0"
git remote add origin https://github.com/사용자이름/quizmate.git
git branch -M main
git push -u origin main
GitHub Pages 기능 활성화:

생성한 GitHub 저장소 페이지의 우측 상단 Settings 메뉴로 이동합니다.
좌측 네비게이션 바에서 Pages 탭을 클릭합니다.
Build and deployment 섹션의 Source 설정을 Deploy from a branch로 선택합니다.
Branch 설정을 main (혹은 master) 가지의 / (root) 폴더로 설정한 후 Save 버튼을 누릅니다.
배포 확인:

설정 후 약 1~2분 뒤 페이지 상단에 배포 완료 알림과 함께 URL(예: https://사용자이름.github.io/quizmate/)이 생성됩니다. 해당 주소로 접속하면 스마트폰이나 태블릿 등 어디서든 사이트를 사용해 학습을 시작할 수 있습니다.
